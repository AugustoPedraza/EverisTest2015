Se requiere generar una aplicación web ASP.net para almacenar y gestionar Skills de los empleados de una compañía de desarrollo y mantenimiento de software.
Un Skill, es una capacidad técnica con un determinado nivel de conocimiento. Esto es, por ejemplo, una persona tiene un Skill intermedio en la tecnología Java.
Requisitos de alto nivel.
  - Se requiere que los empleados se pueden loguear y modificar sus propios skills, es decir, que puede modificar el nivel de conocimiento de cada skill.
  - Se requiere conocer el nivel en las siguientes tecnologías: .NET, Java, PHP, Cobol, Javascript, HTML5, Ruby. Los atributos para las tecnologías son Nombre y Descripción.
  - Se requiere que para cada nivel se pueda evaluar por la experiencia en cada tecnología: Niguna, Básico, Intermedio, Avanzado, Experto.
  - Se requiere que exista un usuario Administrador de la aplicación y que pueda:
  - Añadir/actualizar/borrar empleados.
  - Añadir/actualizar/borrar tecnologías
  - Los datos que se deben almacenar de los empleados son: nombre, ID empleado, teléfono, email, DNI.
  - Lista de empleados iniciales: John Doe, Peter O’Brien, Hans Solo, Michael Moore, James Bond, Julia Roberts, Bruce Dickinson.
